# GitHub Explorer

A React SPA that searches GitHub users via the public REST API and displays their repositories with sorting, filtering, and bookmarking.

## Stack

- React 18 + Vite
- React Router v6
- CSS Modules
- Native `fetch` + `AbortController`
- Context API for cross-cutting state (theme)

## Scripts

```sh
npm install
npm run dev      # start dev server
npm run build    # production build
npm run preview  # preview production build
```
